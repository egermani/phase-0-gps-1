# phase-0-gps-1

`git clone`

    create a new **local** copy of the repo

`touch`

    initialize a **blank** file

`git add`

    add a file to the **staging**

`git commit`

    create a *commit*

`git push`

    send the *commit* up to the remote

`git co -b`

    switch to a different branch. (the b flag creates a new branch.)
